Zone Assignment Rules (applied automatically at seed time)
⚙️ Enter area code prefixes to auto-assign zones and types when seeding. All unmatched locations default to Silver / Standard. You can edit any location individually in the All Locations tab afterward.
💡 Zone rules use AREA:LEVEL format.   Type rules use AREA, AREA:AISLE, or AREA:AISLE:BAY-BAY format.
  • Zone examples: FA:C = Gold   FA:B = Silver   FA:A = Bronze
  • Type examples: FA:09:01-20 = FA aisle 09 bays 01–20 → Crushable   DD = all DD → Pallet Flow
Location ID structure: FA 09 39 D 05 → area=FA, aisle=09, bay=39, level=D, pos=05
Updates zone/type on every location in the database based on current rules above.
